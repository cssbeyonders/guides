==== Tightening and Loosening

- Turn *clockwise* to tighten.
- Turn *anticlockwise* to loosen.
- Use screwdrivers.
- Hold down the nut while tightening/loosening.

[#using-mac]
=== Using Mac

- The command (`Cmd`) key is like the control key on Windows.

+
.Keys on Mac.
[%autowidth]
|===
|Windows |Mac

|`Ctrl` + `C` |`Cmd` + `C`
|`Ctrl` + `V` |`Cmd` + `V`
|===

+
[NOTE]
There is also the control key, but its use is different from that on Windows.

- Change keyboard using the globe key.

- `Control` + `Click` to right click.
