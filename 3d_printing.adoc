[#importing-to-creality-software]
=== Importing to Creality Software

. https://www.creality.com/download#sparkx[Download the software] if you have not.

NOTE: The instruction is based on the version of the Creality software as of 1 April, 2026.

. Import the `.stl` file into the Creality app by pressing the *Add* button.

+
.The "Add" button
image::import_btn.png[Add Button, 300, 200]

+
The model is imported onto the plate, but the orientation may not be correct.

+
.An example of a poor rotation. This would use an excessive amount of support.
image::poor_orientation.png[Poor Orientation Example, 300, 200]

+
[WARNING]
====
A person tried printing out a object without using a proper orientation. The model was placed in a diagonal orientation pointing upward. As a result, the angle make the filament couldn't stick properly to the previous layer and printed in air.

image::tc_poor_orient.jpeg[Poor Orientation Example, 300, 200]
====

. Fix the *rotation* using the *Rotate* feature. It will bring up a pop-up where you can modify the angles by the axis shown by the colored lines.

+
[TIP]
Alternatively, you can use the *Auto Orient* function to minimize the amount of support used.

+
.The location of the "Rotate" and "Auto Orient" buttons on the toolbar
image::rotate_auto_rot.png[Rotate and Auto Orient Buttons, 400, 100]

+
.The interface of the "Rotate" function
image::rot_ui.png[UI of Rotate, 400, 100]

. Fix the *location* by *dragging* the model back onto the plate.

+
[.float-group]
--
[.left]
.An out-of-bound object.
image::obj_out_of_bounds.png[Object Out of Bounds, 200, 300]

[.left]
.A properly placed object.
image::obj_correct.png[Correctly Placed Object, 200, 300]
--

+
NOTE: Sometimes, if the model is too large, it will never fit on the plate, and you need to redesign it.

[#nozzle-and-printer-selection]
=== Nozzle and Printer Selection

You can change the nozzle size and printer in the top left corner.

NOTE: The nozzle size is the diameter of the extruded filament. A smaller number results in a finer object printed.

TIP: It is found out that a smaller nozzle size can also reduce the weight due to reduced wall thickness. Therefore, 0.4 nozzle is better for most situations.

.Selecting printer and nozzle size.
image::printer_nozzle.png[Printer and Nozzle Selection, 300, 200]

. If the list does not have the nozzle and printer you want:
.. Click the *Select/Remove printers*.
.. Check the printers you want to add
.. Click the *+* button for nozzle.
.. Finally, click *Confirm* to save the changes.

+
.The printer list.
image::printer_nozzle_add.png[Printer List, 300, 200]

[TIP]
====
- You can also view the size limit of the printers in the  printer list.
- After selecting a printer and nozzle, the plate will change according to the printer model.
- You might want to reposition your model when the plate changes.
====

. Select the printer and nozzle size that you want to use from the list.

[#multicolor-and-filament-selection]
=== Multicolor and Filament Selection

You can change or add filament type using the *Filament* window on the top right corner.

CAUTION: You cannot set up filament type, multicolor, or printing temperature on the printer; you must set it up in the software.

.The filament window.
image::multifilament.png[Filament Window, 300, 200]

. Add a new filament by pressing *+*.
. Modify the filament's type by pressing the *down arrow* next to the type.
. Change the filament's represented color by pressing the *number* and selecting a color.

+
CAUTION: Changing the color here has nothing to do with the actual printed color. It is only the represented color in the software.

. Next, use the *Color painting* tool to color your model.

+
.The location of the "Color Painting" tool on the toolbar.
image::fill_tool.png[Color Painting Tool, 400, 100]

+
.The UI of the "Color Painting" tool.
image::paint_popup.png[Color Painting UI, 200, 300]

. After filling multiple colors, a prime tower is needed. Make sure your model does not collide with the prime tower.

+
.Prime Tower.
image::prime_tower.png[Prime Tower, 300, 200]

+
However, our printer is terrible. It only supports 1 filament. Filling the surfaces has no effect on the actual color printed. Therefore, we need to <<adding-pauses, add a pause>> during the print and change the filament during the pause.

CAUTION: <<slicing, Slice>> the place first before adding a pause.

NOTE: filling the color makes it easier to identify parts.

[#slicing]
=== Slicing

. If your model has hanging parts, you need to enable support first. On the sidebar, click *Support* > *Enable Support*.

+
.The location of the button of the "Support" tab.
image::enable_support.png[Enable Support, 200, 300]

+
[WARNING]
====
A person forgot to use support for a model with hanging parts. As a result, the hanging part could not print properly.

image::tc_no_support.jpeg[Consequences of Forgoting to Use Support, 300, 200]

Thefore, make sure to enable support when the model have hanging parts.
====

. Click *Slice Plate* (at the bottom right of the canvas).

+
CAUTION: You must slice it first before printing.

+
."Slice Plate" and "Send Print" buttons.
image::slice_print_btn.png[Slice Plate and Send Print buttons, 300, 200]

[#adding-pauses]
=== Adding Pauses

. On the *layer slider*, right click on the layer that you want a pause for.

. Click *Add Pause*.

+
TIP: On Mac, right-clicking on tracepad is clicking with 2 fingers. However, it may trigger scrolling, which changes the layer. You can change the settings so that right-clicking is clicking at the bottom-right corner.

+
NOTE: The pause comes after the layer has finished.

+
.The Layer Slider
image::layer_slider.png[Layer Slier, 300, 200]

[#connection-and-printing]
=== Connection and Printing

. If you have not connected to the printer:

.. Click *Device* on the top toolbar.
.. Click *Scan Add*.

+
."Device" and "Scan Add" buttons
image::connect_printer.png[Device Tab, 300, 200]

.. It will show all the printers. *Tick* the boxes to add them.
.. Click *Add*.


+
[TIP]
====
If none is found:

- go to 211 and check if the printers are on
- check if your computer is connected to css_Internal
====

.. Go back to the *Preview* tab by pressing it on the top of the screen.
. Press *Print Plate*.
. Select the printer.
. Press *Print*.

+
NOTE: Alternatively, you may send the file to the printer and start the printing directly on the printer itself. You can only send when the printer is printing.

+
.The printing screen.
image::send_print.png[Printing Screen, 300, 200]

+
CAUTION: Make sure that the printer selected in this step is the same as the printer when you select the nozzle. Otherwise, the printer may not print correctly or throw errors.

WARNING: A person sent the model which had its printer set to a larger one to a smaller one. As the model was compiled with coordinates for larger sizes, the nozzle went out of range in the smaller printer, resulting in a failed printing job.
